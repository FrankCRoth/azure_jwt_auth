# AzureJwtAuth

WIP

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'azure_jwt_auth'
```

And then execute:

```bash
$ bundle
```

Or install it yourself as:

```bash
$ gem install azure_jwt_auth
```
